**🛍️ Orgs - Gerenciador de Produtos**

Orgs é um aplicativo desenvolvido durante os cursos da Alura para aprendizado e prática de desenvolvimento Android com Kotlin.
O projeto tem como objetivo gerenciar produtos, permitindo cadastrá-los, visualizá-los em uma lista e acessar seus detalhes.

**📱 Tecnologias Utilizadas**

Kotlin - Linguagem principal do projeto               
Android SDK - Desenvolvimento para dispositivos Android                      
Material Design Components - Para a criação de uma UI moderna e responsiva                       
Coil - Biblioteca para carregamento de imagens                   
Room (próximos passos) - Persistência de dados no banco local                    
ViewBinding - Para facilitar a manipulação das Views                    
Intents & Parcelable - Para passagem de dados entre Activities                    

**🎨 Funcionalidades Implementadas**

✅ Listagem de Produtos - Exibe uma lista de produtos cadastrados                 
✅ Cadastro de Produtos - Permite inserir nome, descrição e valor do produto              
✅ Detalhes do Produto - Ao clicar em um item da lista, exibe mais informações                      
✅ Passagem de Dados entre Activities - Utilizando Parcelable                           
✅ Carregamento de Imagens com Coil - Placeholder, fallback e tratamento de erros                     
✅ Formatação de Moeda - Usando NumberFormat para exibir valores no formato brasileiro                    
✅ Delegação de Responsabilidade com AlertDialog - Personalização de alertas                     
✅ Personalização de UI - Uso de componentes do Material Design, como CardView e Floating Action Button                           

**🚀 Próximos Passos**

🔹 Implementação do Room Database - Para salvar os produtos localmente        
🔹 Melhoria na Experiência do Usuário - Ajustes na interface e navegação                      
🔹 Validações no Formulário de Cadastro - Para evitar erros no preenchimento                     
🔹 Melhorias na Performance - Otimização do carregamento da lista de produtos                           

**🛠️ Como Executar o Projeto**

**✅ Pré-requisitos**        

Antes de executar o projeto, certifique-se de ter instalado:                              
Android Studio (versão mais recente)             
JDK 11 ou superior                          
Emulador Android configurado ou um dispositivo físico com modo desenvolvedor ativado  

🔄 Passo a Passo   

1️⃣ Clone o repositório                   
```git clone https://github.com/guing2003/Orgs```

2️⃣ Abra o projeto no Android Studio           
Selecione "Open an Existing Project" e escolha a pasta do projeto clonado               

3️⃣ Instale as dependências             
O Android Studio irá baixar automaticamente as dependências declaradas no build.gradle          
Caso necessário, clique em "Sync Now" na parte superior da tela                      

4️⃣ Configure um dispositivo para teste            
Emulador: Vá até AVD Manager e crie um dispositivo virtual           
Dispositivo físico: Conecte via USB e ative a depuração USB nas configurações do Android            

5️⃣ Execute o projeto       
Clique no botão Run ▶ no Android Studio                   
Escolha o emulador ou dispositivo físico conectado                      
Pronto! O app será instalado e executado no dispositivo escolhido. 🚀
